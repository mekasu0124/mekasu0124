# Desktop Application Projects
- [TrackIt Console](https://github.com/mekasu0124/TrackItConsole)
    - A [C# Console Based Project](https://www.c-sharpcorner.com/article/creating-console-application-in-c-sharp/) that is meant to assist users who have to track their Earnings, Mileage, and Shift Times in a more efficient and appropriate manor. This program uses a SQLite database and also displays totals to the user on the initial home screen. In addition, this program also allows the user to not only create new Shift/Gas/Maintenance records throughout their usage of this program, but also allows the user to Edit/Delete/View any/all records that are stored within the database. The information requested and stored during
    usage of this program is saved locally on the user's computer and is <b>not</b> stored anywhere online. This is to assist the user in keeping their records private and only sharable by the user themselves.

- [Meks Phone Book](https://github.com/mekasu0124/MeksPhoneBook)
    - A [C# Based Application](https://learn.microsoft.com/en-us/dotnet/csharp/tour-of-csharp/) that uses the [Avalonia Framework](https://avaloniaui.net/) to allow the user to store their contacts on their computer. The user is able to create a new contact which allows them to enter the contacts First/Middle/Last name alone with up to 3 emails, phone numbers, and addresses. This project also uses a json file that is queried upon creating a new, or updating an existing, contact to display a combo box drop down of phone number prefixes from all countries within the world that are applicable and then uses that prefix to appropriately format said phone number for proper display and database storage. The information requested and stored during
    usage of this program is saved locally on the user's computer and is <b>not</b> stored anywhere online. This is to assist the user in keeping their records private and only sharable by the user themselves.
